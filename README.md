# Pathfinding Robot
A* Pathfinding Robot on txt mazes

Has graphical/terminal mode

## Utilisation :
```
make
./prog.exe [Carte]
```

#### Contexte 
Pour terminer le cour d'algorithmique avancée en fin d’année de CIR1, nous avons réalisé un algorithme de pathfinding qui permet de résoudre des labyrinthes d’un fichier texte, le programme peut être exécuté depuis un terminal ou depuis une fenêtre graphique crée avec la librairie SDL.
